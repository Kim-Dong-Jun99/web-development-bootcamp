글씨를 이탈릭으로 눕히고 싶거나, 볼드체로 만드록 싶을때는 어떻게 할 수 있을까, 두가지 방법이 있다, <i>, <em> 태그를 사용하는 것인데, 둘 다 글씨를 이탈릭으로 보이게하는 역할을 한다. 그럼 코딩할때 어떤 것을 사용해야할까? 답은 em이다. 그 이유는 i는 단순히 텍스트를 이탈릭으로 보이게만한다. 반면, em, strong 같은 태그는 텍스트에 대한 부가적인 정보도 담고 있어서, strong,em 같은 태그를 활용하는 것이 좋다.

img는 소스파일에 경로를 입력해줘야한다. 아 그리고 alt에는 사진을 디스플레이할 수 없을 시 보여줄 텍스트를 지정할 수 있는데, 그 텍스트는 구글이 검색할때 참고하여서, alt에 키워드를 넣어서 검색될때 노출되게 할 수 있다

이제 하이퍼링크를 넣는 것을 알아보자
하이퍼 텍스트는 텍스트에 링크를 넣어놔서 클릭시 다른 텍스트 문서로 이동할 수 있게 해준다.

테이블은 table 안에, tr를 사용해서 row를 추가할 수 잇고, td로 셀을 추가할 수 있다. 그리고 head, body, footer로 테이블을 분리할 수 잇고, th로 헤더셀을 추가할 수 있다. 이때 바디에 th를 써도 헤더로 인식되는데, 그럼에도 Head, body, footer를 분리하는 이유는 테이블의 영역에 다르게 스타일링할 수 있기 때문이다.

그리고 html에서도 테이블의 스타일을 변경할 수 있다. 그러나 이것은 권장되지 않는다. html은 뼈대, css로 스타일, js로 동적인 역할을 수행하게 코딩하는 것이 좋다

Html 폼으로 메일을 보낼 수도 있고, 특정 Html에 데이터를 보낼 수도 있는 것 같다. name에 값을 할당하면 딕셔너리처럼 값이 매핑된다!

css 등장 배경
레이아웃이 복잡해질 수록 코드를 유지보수하기 어려워진다
그래서 CSS가 등장하였다.

html은 결국 직사각형들의 모음이다! 그래서 css를 통해서 스타일을 각각 변경할 수 있고, 모든 태그에 공통적으로 적용된다. hr 같은 경우, 디폴트로 세팅된 border-style이 있어서 background-color = white로 하면 바로 적용 되지 않는다. 보더 스타일 값을 none으로 설정해줘야지 선의 색상이 변경된다

css에서도 직사각형의 top,bottom, left, right을 각각 설정해줄 수 있다

css를 디버깅해보았는데, css로 스타일을 정할때는 외부 css 경로를 확인해야하고, 외부 css에서 스타일을 정의해도, 로컬에서 스타일을 정의하면 스타일이 바뀔 수 있으니, 디버깅할때 참고하면 좋을 것 같다

css로 스타일을 정의할때, 태그 뿐만아니라, 클래스를 명시해서 같은 태그이더라도, 다른 스타일링을 할 수 있다.

아이디로 css 스타일링을 할 수도 있는데, 아이디 설렉터는 쓰지 말라는 말이 콘솔에서 나온다. 아이디는 유니크해야한다! 태그 설렉터로 하면 여러개의 스타일을 적용시킬 수 있고, 중복해서도 사용이 가능하다. 아 그리고 브라우저에서 디폴트로 적용 시키는 스타일 시트도 존재한다.

pseudo 클래스
몇개의 태그는 :이 이름앞에 붙어있는 것을 확인할 수 있다, 이것들은 pseudo 클래스들이다! html 엘리멘트들이 다른 상태에 놓여있을 수 있기에, 이런 클래스들이 존재한다. css 에 hover를 추가해보자. 예제 코드
img:hover{
  background-color: gold;
}
